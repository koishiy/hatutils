## Hatser

Build the auto splitter with ./build.sh

Once built all you have to do is run it (preferably in the console).

Go to LiveSplit One and connect to the url: `ws://localhost:16834` (you can change this if you desire).

Once connected and the cli says it's attached, you should be good to go.

This is still very beta so ping me in disccord if something doesn't work lol.
